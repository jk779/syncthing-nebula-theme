# Bundled font licenses

The fonts in this directory are bundled locally so the Nebula theme does not
depend on external font requests at runtime.

- Google Sans Flex — SIL Open Font License 1.1, obtained from Google Fonts.
- Stack Sans Notch — SIL Open Font License 1.1, obtained from Google Fonts.
- DM Mono — SIL Open Font License 1.1, obtained from Google Fonts.

The font files are unmodified Google Fonts webfont builds.
